# Block 4 Isolated-Note Lineage Snapshot

Status: `ongoing research`, but already strong enough as a public scientific
checkpoint.

This example is the Block 4 public entry point for isolated real notes.

It is designed to show, in the smallest useful package, that the method is not
just naming notes or drawing decorative spirals. It is trying to recover the
causal resonance life of a real note.

## Goal

Show three things at once:

1. a real isolated piano note contains structured harmonic lineage rather than a
   clean tone plus random tail;
2. different instruments playing the same note region show different harmonic
   amplitude behavior;
3. `unassigned resonance` should be treated as unresolved causal structure, not
   as empty noise.

## Why this example matters

This is the smallest public Block 4 package that already demonstrates:

- note-level harmonic lineage;
- note-box and residual growth around a real note;
- cross-instrument morphology difference;
- a non-trivial interpretation of residual and unassigned layers.

## Key observations in this snapshot

From the first real piano note `001_piano_real_5.A-`:

- total lineage points: `25244`
- harmonic core points: `267`
- spawned note-box points: `2695`
- spawned residual points: `2937`
- unassigned points: `19345`

From the cross-instrument morphology compare on `9.5-`:

- `RealPiano_1_1 vs cello` morphology distance: `0.4218`
- `RealPiano_1_1 vs violin` morphology distance: `0.4131`
- `cello vs violin` morphology distance: `0.2920`

This is a useful public checkpoint because it shows that:

- the note has structured internal growth;
- strings are morphologically closer to each other than either is to piano;
- the unresolved layer remains large even in controlled isolated-note material.
