# Examples

This directory is for small, reproducible public examples only.

Recommended first public cases:

1. One-note cross-instrument comparison
2. Short polyphonic fragment with event-count alignment
3. Same-note / different-instrument separation case

Each example should contain:

- a short audio fragment
- a small CSV or JSON summary
- one image or HTML visualization when useful
- a short `HOW_TO_RUN.md`

Do not place full archives or long research runs here.
