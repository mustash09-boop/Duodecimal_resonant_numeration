# Same Note: Banjo vs Guitar

This is the first compact public example for the project.

## Goal

Show that the project is not limited to note naming alone. Two instruments can
play the same note while producing different resonance structure.

## Note

- canonical note: `8.8-`
- musical naming in source files: `G3`

## Included files

- `audio/banjo_G3_very-long_forte_normal.wav`
- `audio/guitar_G3_very-long_forte_normal.wav`
- `compare/note_compare__8.8-.csv`
- `compare/note_compare__8.8-.md`

## Why this example matters

This is a minimal case for:

- same note
- different instruments
- different body behavior
- different resonance field

It is a good public entry point before moving to denser polyphonic cases.
