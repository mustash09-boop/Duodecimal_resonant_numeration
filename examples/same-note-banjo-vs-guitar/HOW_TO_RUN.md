# How To Run

This example is intentionally small and mostly inspection-oriented.

## Step 1

Listen to the two source notes:

- `audio/banjo_G3_very-long_forte_normal.wav`
- `audio/guitar_G3_very-long_forte_normal.wav`

## Step 2

Read the compact comparison:

- `compare/note_compare__8.8-.md`

## Step 3

Inspect the raw numeric comparison:

- `compare/note_compare__8.8-.csv`

## What to look for

- different chain density
- different instrument-body behavior
- different time span of box response
- same note identity but different resonance structure
