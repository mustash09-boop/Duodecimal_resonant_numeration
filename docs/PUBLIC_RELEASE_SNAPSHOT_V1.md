the project.

- main `music12` package
- `Block002_pipeline`
- `Block002_audio_recogn`

1. `same-note-banjo-vs-guitar`

## Public positioning

This repository should be read as an emerging research lab.

It contains:

- core code
- selected tools
- compact examples
- public-facing documents

It does not attempt to place all raw datasets, large scans, long WAV outputs,
or machine-local research traces into GitHub.

## Current research honesty

The repository intentionally includes examples marked as `ongoing research`.
They are meant to show:

- what the framework already sees clearly
- where event structure is already interpretable
- where instrument-role reasoning is emerging
- where full separation remains unresolved
